components/  Hiển thị giao diện tái sử dụng giao diện UI (header, footer, menu, ...)
services/    Chỉ định nghĩa cách kết nối dữ liệu (Backend). (data access layer)
hooks/       Gom logic xử lý (State/Effect) để dùng lại. (quản lý state + lifecycle)
context/     Lưu trữ dữ liệu dùng chung cho cả app.
pages/       Hiển thi giao diện các trang (Home, Detail, Cart, Checkout,...)
types/       Định nghĩa interface
